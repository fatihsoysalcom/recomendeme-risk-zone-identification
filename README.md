# RecomendeMe Risk Zone Identification

This example simulates 'RecomendeMe Intelligence' by analyzing a dataset of various incidents across Brazil. It identifies potential human trafficking risk zones by clustering incidents and assigning weighted risk scores based on their type and proximity. The output highlights high-priority areas for investigation, demonstrating how data analysis can 'map the darkness' and aid in combating human trafficking.

## Language

`python`

## How to Run

Save the code as `main.py`. Open a terminal or command prompt, navigate to the directory where you saved the file, and run `python main.py`.
